# 📚 BookTalk (ブックトーク)

「今読み終わった本について、勢いそのまま！感動を語り合いたいんだど、周りに読んだ人がいない…誰か同じ本を読んだ人と話がしたい！」
そんな悩みを解決する、**書籍検索＆即時ビデオ通話アプリ**です。

## 🚀 特徴
* **即時検索:** Google Books APIを使用し、世界中の本を検索可能。
* **ワンクリック通話:** Jitsi Meet APIを埋め込み、登録不要でビデオ通話を開始。
* **完全レスポンシブ:** スマホ・PC問わず、URL一つで利用可能。

## 🛠 使用技術
* **言語:** Python 3.x
* **フレームワーク:** Streamlit
* **API:** Google Books API, Jitsi Meet
* **インフラ:** Streamlit Cloud

## 💡 工夫した点
* 検索結果からシームレスに通話画面へ移行するUXを設計。
* ユーザーが選んだ本のIDに基づいて部屋を生成し、混線を防止。
* モバイル利用時のカメラ権限問題を回避するため、別タブ誘導ボタンを実装。
